# A3D-History

Разработка началась в 1998м (весной), с самого начала проект был запланирован в 3д.

Релиз:
Россия, Германия - 26 октября 2000 года.

Этапы:
1. преальфа - этап начальной разработки движка - город, лучница, маг, сражение с панцирниками
2. альфа98 - тёмнозелёная трава, два посёлка
3. альфа99 - с зимы по лето 1999 - светлозелёная (салатовая) трава
4. альфа00 - с лета 2000 - привычный вид ландшафта
5. бета - поздний этап разработки, перед релизом
